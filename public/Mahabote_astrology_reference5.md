# မဟာဘုတ်ဗေဒင်ပညာ — အခြေခံလက်စွဲ

> မြန်မာ့ရိုးရာ မဟာဘုတ်ဗေဒင်ပညာအကြောင်း စုစည်းထားသော ကိုးကားစာရွက်။
> အရင်းအမြစ် — Myanmar Wikipedia, SharingKnowledges, CompuBaydaThukhuma, MyKyawZaya Blog စသည်။

---

## ၁။ အဓိပ္ပာယ်နှင့် မူလအစ

**မဟာဘုတ်** = မဟာ (ကြီးမြတ်သော) + ဘူတ (ဖြစ်ခြင်း)

ကျယ်ဝန်းနက်ရှိုင်းသော အခြေခံဓာတ်ကြီးများဟု အဓိပ္ပာယ်ရသည်။ အဘိဓမ္မာတွင် မဟာဘူတရုပ် (၄) ပါး — **ပထဝီ၊ အာပေါ၊ တေဇော၊ ဝါယော** — ကိုလည်း မဟာဘုတ်ဟု ခေါ်သည်။

- သမားတော်ကြီး **ဇီဝက** မှ စတင်ခဲ့သည်ဟု အဆိုရှိသည်
- **ဆေးပညာမဟာဘုတ်** နှင့် **ဗေဒင်ပညာမဟာဘုတ်** ဟူ၍ နှစ်မျိုးရှိသည်
- **အားသာချက်** — မွေးဖွားချိန် (နာရီ/မိနစ်) သိရန်မလိုဘဲ နေ့နံနှင့် မွေးနှစ်သာလျှင် တွက်နိုင်ခြင်း

---

## ၂။ အိမ် ၇ လုံး (ဌာန ၇ ခု)

| အစီ | အိမ် | သဘော | အဓိပ္ပာယ် |
|---|---|---|---|
| ၁ | **အထွန်း** | အထက်တန်း | ထွန်းပေါက်၊ ကျော်ကြား၊ ရုပ်ရည်လှ |
| ၂ | **သိုက်** | အထက်တန်း | ဥစ္စာပစ္စည်း၊ အိမ်ခြံမြေ၊ ကြေးငွေ |
| ၃ | **ရာဇ** | အထက်တန်း | အာဏာ၊ ရာထူး၊ အောင်မြင်မှု |
| ၄ | **အဓိပတိ** | အထက်တန်း | ခေါင်းဆောင်၊ ဩဇာ၊ တိုးတက်မှု |
| ၅ | **ပုတိ** | အောက်တန်း | ပုပ်ပျက်၊ ခြူခြာ၊ ငယ်ဇာတာမကောင်း |
| ၆ | **ဘင်္ဂ** | အောက်တန်း | ပြိုလဲ၊ ပျက်စီး၊ အတတ်အကျများ |
| ၇ | **မရဏ** | အောက်တန်း | သေခြင်း၊ ရောဂါ၊ ဆုံးရှုံးခြင်း |

### အိမ်စီစဉ် (လက်ယာရစ်)

```
ဘင်္ဂ → မရဏ → အထွန်း → သိုက် → ရာဇ → ပုတိ → အဓိပတိ
```

> **မှတ်ချက်** — အထက်တန်းဖွားဖြစ်ရုံဖြင့် အမြဲအကောင်းမဟုတ်၊ အောက်တန်းဖွားဖြစ်ရုံဖြင့်လည်း အမြဲအဆိုးမဟုတ်ပါ။ ကျော်ကြားသူအများစုတွင် နှစ်ဖက်လုံး အရေအတွက် မတိမ်းမယိမ်းရှိနေသည်။

## ၃။ ဂြိုဟ် ၇ လုံးနှင့် ဂဏန်းကိန်း

| နေ့နံ | ဂဏန်း | ကိန်း (အက္ခရာ) |
|---|---|---|
| တနင်္ဂနွေ | ၁ | အောင် |
| တနင်္လာ | ၂ | ကြီး |
| အင်္ဂါ | ၃ | စစ် |
| ဗုဒ္ဓဟူး / ရာဟု | ၄ | လံ |
| ကြာသပတေး | ၅ | ပွဲ |
| သောကြာ | ၆ | သူ |
| စနေ | ၀ / ၇ | ထူ |

### မှတ်ရန်လွယ်သော ကဗျာ

> **"အောင် လံ ထူ စစ် သူ ကြီး ပွဲ"**
> **" ၁   ၄   ၇   ၃   ၆   ၂   ၅ "**

---

## ၄။ ဖွားဇာတာ တွက်နည်း

### အဆင့်

1. မြန်မာသက္ကရာဇ်ကို တည်ထား
2. ၇ ဖြင့် စား → အကြွင်းယူ
3. အကြွင်းနှင့်တူသော နေ့နံကိန်း (အောင်/လံ/ထူ...) ကို **ဘင်္ဂအိမ်မှ စတင်၍** ချ
4. ကျန်ကိန်းများကို **လက်ယာရစ်** အတိုင်း ဆက်ချ (ဘင်္ဂ → မရဏ → အထွန်း → သိုက် → ရာဇ → ပုတိ → အဓိပတိ)
5. **မိမိမွေးနံကိန်း** ကျရောက်သောအိမ်သည် **ဖွားဇာတာ**

### ဥပမာ တွက်ပြခြင်း

**၁၃၄၂ ခုမွေး တနင်္လာသား**

- ၁၃၄၂ ÷ ၇ = ၅ ကြွင်း
- ၅ ကြွင်း = ကြာသပတေး = **"ပွဲ"**
- ဘင်္ဂအိမ်မှ စ၍ လက်ယာရစ်ချ:

| အိမ် | ကိန်း | နေ့နံ |
|---|---|---|
| ဘင်္ဂ | ပွဲ | ကြာသပတေး |
| မရဏ | အောင် | တနင်္ဂနွေ |
| အထွန်း | လံ | ဗုဒ္ဓဟူး |
| သိုက် | ထူ | စနေ |
| ရာဇ | စစ် | အင်္ဂါ |
| ပုတိ | သူ | သောကြာ |
| **အဓိပတိ** | **ကြီး** | **တနင်္လာ** |

→ တနင်္လာသားသည် **အဓိပတိအိမ်** တွင်ကျ၍ **အဓိပတိဖွား** ဖြစ်သည်။

### အကြွင်းမှ တိုက်ရိုက်ရှာနည်း (နေ့သားအလိုက်)

**တနင်္ဂနွေ သားသမီး**
| အကြွင်း | ဖွားဇာတာ |
|---|---|
| ၁ | ဘင်္ဂ |
| ၂ | အထွန်း |
| ၃ | ရာဇ |
| ၄ | အဓိပတိ |
| ၅ | မရဏ |
| ၆ | သိုက် |
| ၀/၇ | ပုတိ |

**တနင်္လာ သားသမီး**
| အကြွင်း | ဖွားဇာတာ |
|---|---|
| ၁ | ပုတိ |
| ၂ | ဘင်္ဂ |
| ၃ | အထွန်း |
| ၄ | ရာဇ |
| ၅ | အဓိပတိ |
| ၆ | မရဏ |
| ၀/၇ | သိုက် |

*(အခြားနေ့နံများအတွက်လည်း အလားတူပုံစံဖြင့် တွက်နိုင်သည်။)*

### ခရစ်နှစ်မှ မြန်မာသက္ကရာဇ်သို့ ပြောင်း

- **မြန်မာနှစ်ဆန်း (ဧပြီ ၁၇ ခန့်) နောက်ပိုင်းမွေး** → ခရစ်နှစ် − ၆၃၈
- **မြန်မာနှစ်ဆန်း မတိုင်မီမွေး** → ခရစ်နှစ် − ၆၃၉

---

## ၅။ ဖွား ၇ မျိုး × အိမ် ၇ မျိုး = ၄၉ ဇာတာ

ထပ်၍ **လူနက္ခတ်၊ နတ်နက္ခတ်၊ ဘီလူးနက္ခတ်** ဟူ၍ ခွဲနိုင်သေးသည်။

---

## ၆။ ဖွား ၇ မျိုး၏ ယေဘုယျ အကျင့်

### အထက်တန်းဖွား

- **အထွန်းဖွား** — လှပ၊ ပေါ့ပါး၊ ခရီးသွားလာကြိုက်၊ အိမ်မကပ်စီးပွား အောင်မြင်
- **သိုက်ဖွား** — လက်တွေ့ကျ၊ ပင်ပန်းခံ၊ ငွေကိုစနစ်တကျသုံး၊ မိသားစုအတွက် ငွေကုန်များ
- **ရာဇဖွား** — အာဏာရ၊ ခေါင်းဆောင်ပီသ၊ ဂုဏ်သိက္ခာကြီးမား
- **အဓိပတိဖွား** — ဉာဏ်ပညာထက်မြက်၊ တိုးတက်မှုဖြောင့်ဖြူး

### အောက်တန်းဖွား

- **ပုတိဖွား** — တာဝန်ကျေပွန်လို၊ ငယ်စဉ်ကံနည်း၊ ရှေ့အစောပိုင်းခက်ခဲ၊ အခွင့်ထူးခံအလုပ်မမုန်
- **ဘင်္ဂဖွား** — တစိမ်းထက် ဆွေမျိုးက ဒုက္ခပေး၊ ဇာတိအဝေးမှာ ထွန်းပေါက်၊ ကောက်ရိုးမီးစိတ်
- **မရဏဖွား** — စိတ်ပြင်းထန်၊ ထက်မြက်၊ ချိတုံချတုံ၊ နှစ်ရှည်လုပ်ငန်းနှင့်မသင့်၊ စေ့ငုလုပ်တတ်၊ မေ့လွယ်၊ လောကဓံခံနိုင်

---

## ၇။ ဘုမ္မိ (အားနည်းသော အချိန်ပိုင်း)

ဘုမ္မိ = ဘဝ၏ **အားနည်းသော/အန္တရာယ်ကျရောက်တတ်သော** အသက်နှစ်များ။

### ၇.၁ ဘုမ္မိလေး (Small Bhummi)

မိမိမွေးနံမှ ၇ ခုမြောက်နေရာ။ **အသက် ÷ ၈ = ၇ ကြွင်း** သော နှစ်များ။

**နေ့သားအားလုံး** — ၇၊ ၁၅၊ ၂၃၊ ၃၁၊ ၃၉၊ ၄၇၊ ၅၅၊ ၆၃ ...

### ၇.၂ ဘုမ္မိ (နေ့သားအလိုက်)

| နေ့သား | ဘုမ္မိနှစ် |
|---|---|
| တနင်္ဂနွေ | ၃၆၊ ၄၄ |
| တနင်္လာ | ၄၄ |
| အင်္ဂါ | ၃၆၊ ၄၄၊ ၅၂ |
| ဗုဒ္ဓဟူး | ၅၂ |
| ကြာသပတေး | ၅၂ |
| သောကြာ | ၄၄ |
| စနေ | ၄၄၊ ၅၂၊ ၆၀ |

### ၇.၃ ဘုမ္မိကြီး (Great Bhummi — အပြင်းထန်ဆုံး)

| နေ့သား | ဘုမ္မိကြီးနှစ် |
|---|---|
| တနင်္ဂနွေ | ၃၁၊ ၃၉ |
| တနင်္လာ | ၄၇ |
| အင်္ဂါ | ၃၉၊ ၄၇ |
| ဗုဒ္ဓဟူး | ၄၇၊ ၅၅ |
| ကြာသပတေး | ၅၅ |
| သောကြာ | ၄၇ |
| စနေ | ၄၇၊ ၅၅ |
| ရာဟု | ၃၉၊ ၄၇ |

### ၇.၄ ဘုမ္မိနှစ်၌ သတိထားရမည့်အရာ

- ကျန်းမာရေး၊ စီးပွားရေး၊ စိတ်ဓာတ်တို့ ထိခိုက်လွယ်
- ဆုံးဖြတ်ချက်ကြီးများ (အိမ်ထောင်ပြု၊ ရင်းနှီးမြှုပ်နှံ၊ ခရီးဝေး) ရှောင်သင့်
- **ယတြာ** ချေပခြင်း — ဒါန၊ သီလ၊ ဘာဝနာ၊ ပရိတ်တရားနာ၊ ဆွမ်းကပ်၊ သက်သတ်လွှတ်

---

## ၈။ ရာဟုနှင့် ဇီဝ

- မဟာဘုတ်အိမ် ၇ လုံးတွင် **ရာဟုကို မထည့်သွင်း** ဘဲ ဗုဒ္ဓဟူးနေ့နံနှင့် တွဲသုံးသည်
- ဗုဒ္ဓဟူးနေ့ **ညနေ ၆ နာရီနောက်ပိုင်းမွေးသူ** ကို **ရာဟုသား** ဟု သတ်မှတ်သည်
- **Birth Lord** (ဇာတာပိုင်ဂြိုဟ်) တွက်ရာတွင် ရာဟုကို ထည့်တွက်သည်

---

## ၉။ ဆက်စပ်ပညာရပ်များ

### ၉.၁ နက္ခတ်တာရာ ၉ မျိုး (Vedic မူ)

| အစီ | နာမည် | အနက် |
|---|---|---|
| ၁ | ဇန္မ | မွေးကြယ် (သာမန်) |
| ၂ | သမ္ပတ် | စည်းစိမ် |
| ၃ | **ဗိပါတ်/ဗျတိ** | ဘေးရန်၊ အန္တရာယ် |
| ၄ | ကျေမ | ငြိမ်းချမ်း |
| ၅ | ပြတျရိ | အတားအဆီး |
| ၆ | သာဓက | အောင်မြင်မှု |
| ၇ | ဝဓ/မရဏ | သေဘေး၊ အပျက် |
| ၈ | မိတ္တ | မိတ်ဆွေ |
| ၉ | အတိမိတ္တ | မိတ်ဆွေအကြီး (အကောင်းဆုံး) |

**အားနည်းဌာန ၃ ခု** — ဗိပါတ် (၃)၊ ပြတျရိ (၅)၊ ဝဓ/မရဏ (၇)။

### ၉.၂ ဥစ္စ / နိစ် (Vedic)

- **ဥစ္စ (Uccha)** — ဂြိုဟ်၏ အမြင့်ဆုံး၊ အားအကောင်းဆုံးနေရာ
- **နိစ် (Neecha)** — ဂြိုဟ်၏ အနိမ့်ဆုံး၊ အားအနည်းဆုံးနေရာ
- **သွေ (Swakshetra)** — ဂြိုဟ်၏ ကိုယ်ပိုင်အိမ်

## ၁၀။ ကသစ်မဟာဘုတ် (အဆင့်မြင့် မဟာဘုတ်)

### ၁၀.၁ ကသစ်မဟာဘုတ် ဆိုသည်မှာ

ကသစ်မဟာဘုတ်သည် မူလမဟာဘုတ် (မွေးနှစ်ဖြင့်တွက်ခြင်း) ၏ အဆင့်မြင့်အပိုင်းဖြစ်သည်။ **ရောက်ဆဲ** (ယခု/လက်ရှိ) **နှစ်သက္ကရာဇ်** ကို ၇ ဖြင့်စား၍ ထွက်လာသော မဟာဘုတ်ကို ကသစ်မဟာဘုတ် (Current Year Mahabote / Transit Mahabote) ဟု ခေါ်သည်။

> မူလမဟာဘုတ် = မိမိမွေးသက္ကရာဇ် ÷ ၇ (ဘဝတစ်သက်တာ ဇာတာ — ပုံသေ)
> ကသစ်မဟာဘုတ် = ရောက်ဆဲနှစ်သက္ကရာဇ် ÷ ၇ (ထိုနှစ်အတွက် ယာယီဇာတာ — နှစ်စဉ်ပြောင်း)

### ၁၀.၂ ကသစ်မဟာဘုတ်၏ သဘော

- **မူလမဟာဘုတ်** သည် လူတစ်ဦး၏ ဘဝတစ်သက်တာ သဘာဝစရိုက်ကို ပြသည်
- **ကသစ်မဟာဘုတ်** သည် ယခုနှစ်အတွက် ဂြိုဟ်အနေအထားပြောင်းလဲမှုကို ပြသည်
- နှစ်ခုပေါင်း၍ ဖတ်မှ ယခုနှစ်တွင် ကံဘယ်လို ရောက်မည်ကို တိကျစွာဟောနိုင်သည်

### ၁၀.၃ ကသစ်၏ ကဗျာလင်္ကာ

> **"ဆင့် မြို့ သား အ တော လှ ကြွား"**

ကသစ်ကို ဘင်္ဂအိမ်မှ ၄ ကြိမ်တိုးကိန်း (ဓာတ်ကြီး ၄ ပါး) ဖြင့် ဇာတာခွင်တွင် ထည့်သွင်းရသည်။

### ၁၀.၄ တွက်နည်း

1. **ရောက်ဆဲနှစ်** (ဥပမာ ၁၃၈၇) ကို ၇ ဖြင့်စား → အကြွင်းယူ
2. မူလမဟာဘုတ်နည်းအတိုင်းပင် ဘင်္ဂအိမ်မှ ကိန်းချ
3. ထွက်လာသော ကသစ်ခွင်ကို မူလခွင်နှင့် ထပ်တင်ကြည့်

### ၁၀.၅ ကသစ်ကြည့်ရာတွင် အဓိကအချက်

- မိမိ**မွေးနံ** သည် ကသစ်ခွင်တွင် **ဘယ်အိမ်** ရောက်နေသလဲ ကြည့်ရန်
- အထွန်း/သိုက်/ရာဇ/အဓိပတိ ရောက်လျှင် — ယခုနှစ် **ကံကောင်း**
- ပုတိ/ဘင်္ဂ/မရဏ ရောက်လျှင် — ယခုနှစ် **သတိထားရန်**
- ကသစ်ခွင်၌ မိမိမွေးနံနှင့် **ဘုမ္မိ** ထပ်မိလျှင် အလွန်သတိထားရ

## ၁၁။ ဂြိုဟ်စီးဂြိုဟ်နင်း

### ၁၁.၁ ဂြိုဟ်စီး ဆိုသည်မှာ

မဟာဘုတ်ခွင်ကို ချပြီးသောအခါ အိမ် ၇ လုံးတွင် ဂြိုဟ် (နေ့နံကိန်း) ၇ လုံး အသီးသီး **ထိုင်နေပြီးသား** ဖြစ်သည်။

- **စီးခြင်း** — ဂြိုဟ် A သည် ဂြိုဟ် B ၏ မူလနေရာ (အိမ်) တွင် ရောက်ရှိနေခြင်း
- **နင်းခြင်း** — အားကောင်းသော ဂြိုဟ်က အားနည်းသော ဂြိုဟ်၏ အိမ်ကို ဖိနှိပ်ခြင်း

ဂြိုဟ်စီးသည် **မူလခွင်** (ဘဝတစ်သက်တာ) နှင့် **ကသစ်ခွင်** (ယခုနှစ်) နှစ်မျိုးလုံးတွင် ဖတ်ရသည်။

### ၁၁.၂ ဂြိုဟ်အင်အား အဆင့်

| အဆင့် | ဂြိုဟ် | သဘော |
|---|---|---|
| **အားကောင်း** | တနင်္ဂနွေ (နေ)၊ ကြာသပတေး (ကြာ) | ဖိနှိပ်နိုင်စွမ်း ကြီး |
| **အားသင့်** | တနင်္လာ (လ)၊ ဗုဒ္ဓဟူး (ဟူး)၊ သောကြာ (သော) | အိမ်သဘောပေါ်မူတည် |
| **အားဆိုး** | အင်္ဂါ (ဂါ)၊ စနေ (နေ)၊ ရာဟု | ဆိုးအိမ်ရောက်လျှင် ထပ်ဆိုး |

### ၁၁.၃ ဂြိုဟ်စီး အကျိုးသက်ရောက်မှု ဇယား

| ဂြိုဟ်အင်အား | စီးနေသောအိမ် | ရလဒ် | ဥပမာ |
|---|---|---|---|
| အားကောင်း | ✅ ကောင်းအိမ် | **ထူးကောင်း** | ကြာ→အထွန်း = ပညာဖြင့်ကျော်ကြား |
| အားကောင်း | ❌ ဆိုးအိမ် | **အဆိုးလျော့** | နေ→မရဏ = ရောဂါရှိသော်လည်း ခံနိုင်ရည်ရှိ |
| အားသင့် | ✅ ကောင်းအိမ် | **ကောင်း** | ဟူး→အထွန်း = ပညာ+ကျော်ကြားမှု |
| အားသင့် | ❌ ဆိုးအိမ် | **ကြားနေ** | သော→ပုတိ = ပျက်စီးမှုကို ကျေနပ်မှုဖြင့် ပြေလည်စေ |
| အားဆိုး | ✅ ကောင်းအိမ် | **ကောင်းယိုယွင်း** | စနေ→သိုက် = ငွေကြေးကျပ်တည်း |
| အားဆိုး | ❌ ဆိုးအိမ် | **⚠ အန္တရာယ်ကြီး** | ဂါ→ပုတိ = ကျန်းမာရေး အထူးသတိ |

### ၁၁.၄ ဂြိုဟ်စီး အသက်ပိုင်း

| ဂြိုဟ် | အုပ်စိုးသော အသက်ပိုင်း | သဘော |
|---|---|---|
| တနင်္လာ (လ) | မွေးစ – ၁ နှစ် | ငယ်ဇာတာ — မိခု |
| အင်္ဂါ (ဂါ) | ၁ – ၂ နှစ် | စတင်လမ်းလျှောက်/ပြေးချိန် |
| ဗုဒ္ဓဟူး (ဟူး) | ၂ – ၈ နှစ် | ပညာသင်ကြားချိန် |
| စနေ (နေ) | ၈ – ၁၅ နှစ် | ကြီးထွားရင့်ကျက်ချိန် |
| ကြာသပတေး (ကြာ) | ၁၅ – ၂၁ နှစ် | ပညာအမြင့်ဆုံး/ဆုံးဖြတ်ချက်ချချိန် |
| တနင်္ဂနွေ (နေ) | ၂၁ – ၃၅ နှစ် | စီးပွား/မိသားစု တည်ဆောက်ချိန် |
| သောကြာ (သော) | ၃၅+ နှစ် | ရင့်ကျက်/အသီးအပွင့်ဆွတ်ချိန် |

> **ဖတ်နည်း** — ထိုအသက်ပိုင်းကို အုပ်စိုးသော ဂြိုဟ်သည် မူလခွင်၌ **ဘယ်အိမ်** စီးနေသလဲ ကြည့် → အကောင်းအဆိုး ဆုံးဖြတ်

### ၁၁.၅ မူလ ဂြိုဟ်စီး — လက်တွေ့ ဥပမာ

**ဥပမာ — ၁၃၄၂ ခုမွေး တနင်္လာသား**

တွက်ချက် — ၁၃၄၂ ÷ ၇ = **၅ ကြွင်း** → ဘင်္ဂအိမ်မှ "ပွဲ" စချ
ကိန်းချစဉ် (+3 mod 7) — **ပွဲ(၅), အောင်(၁), လံ(၄), ထူ(၇), စစ်(၃), သူ(၆), ကြီး(၂)**

| အိမ် | ကိန်း | ဂြိုဟ် | အိမ်သဘော | ဂြိုဟ်အား | ဖတ်ချက် |
|---|---|---|---|---|---|
| ဘင်္ဂ | ပွဲ (၅) | ကြာသပတေး | ❌ ဆိုး | အားကောင်း | ✅ ဆိုးအိမ်ကို ဖိနှိပ် → ပျက်စီးမှု လျော့ |
| မရဏ | အောင် (၁) | တနင်္ဂနွေ | ❌ ဆိုး | အားကောင်း | ✅ ရောဂါဘေးရှိသော်လည်း ခံနိုင်ရည်ရှိ |
| အထွန်း | လံ (၄) | ဗုဒ္ဓဟူး | ✅ ကောင်း | အားသင့် | ✅ ပညာဖြင့် ထွန်းပေါက် |
| သိုက် | ထူ (၇) | စနေ | ✅ ကောင်း | အားဆိုး | ⚠ ဥစ္စာအိမ်ကို ဆိုးဂြိုဟ်စီး → **ငွေကြေးသတိထား** |
| ရာဇ | စစ် (၃) | အင်္ဂါ | ✅ ကောင်း | အားဆိုး | ⚠ ရာထူးအိမ်ကို ဆိုးဂြိုဟ်စီး → **ပဋိပက္ခ** |
| ပုတိ | သူ (၆) | သောကြာ | ❌ ဆိုး | အားသင့် | ↔ ပျက်စီးမှုကို ကျေနပ်မှုဖြင့် ပြေလည်စေ |
| **အဓိပတိ** | **ကြီး (၂)** | **တနင်္လာ** | ✅ ကောင်း | အားသင့် | ✅ **ဖွားအိမ်ကောင်း — ခေါင်းဆောင်ဩဇာ** |

**အသက်ပိုင်းနှင့် တွဲဖတ်**

| အသက်ပိုင်း | အုပ်စိုးဂြိုဟ် | စီးနေသောအိမ် | ဖတ်ချက် |
|---|---|---|---|
| မွေးစ – ၁ နှစ် | တနင်္လာ | **အဓိပတိ** ✅ | ငယ်ဇာတာ ကောင်း |
| ၁ – ၂ နှစ် | အင်္ဂါ | **ရာဇ** ✅⚠ | ကြီးထွားရေးကောင်း / ထိခိုက်ဒဏ်ရာ သတိ |
| ၂ – ၈ နှစ် | ဗုဒ္ဓဟူး | **အထွန်း** ✅ | ပညာသင်ကြားရေး အလွန်ကောင်း |
| ၈ – ၁၅ နှစ် | စနေ | **သိုက်** ⚠ | မိသားစု ငွေကြေးကျပ်တည်းနိုင် |
| ၁၅ – ၂၁ နှစ် | ကြာသပတေး | **ဘင်္ဂ** ✅ | ဘင်္ဂအဆိုးကို ကြာဂြိုဟ်ဖိနှိပ် |
| ၂၁ – ၃၅ နှစ် | တနင်္ဂနွေ | **မရဏ** ⚠ | ကျန်းမာရေး စိန်ခေါ်မှု / ခံနိုင်ရည်ရှိ |
| ၃၅+ နှစ် | သောကြာ | **ပုတိ** ↔ | ပျက်စီးမှုကို ကျေနပ်မှုဖြင့် ကျော်လွှား |

### ၁၁.၆ ကသစ် ဂြိုဟ်စီး — ယခုနှစ် ဥပမာ

**အတူတူ တနင်္လာသား — ယခုနှစ် ၁၃၈၈**

တွက်ချက် — ၁၃၈၈ ÷ ၇ = **၂ ကြွင်း** → ဘင်္ဂအိမ်မှ "ကြီး" စချ
ကိန်းချစဉ် — **ကြီး(၂), ပွဲ(၅), အောင်(၁), လံ(၄), ထူ(၇), စစ်(၃), သူ(၆)**

| အိမ် | ကိန်း | ဂြိုဟ် | အိမ်သဘော | ဂြိုဟ်အား | ဖတ်ချက် |
|---|---|---|---|---|---|
| **ဘင်္ဂ** | **ကြီး (၂)** | **တနင်္လာ** | ❌ ဆိုး | အားသင့် | ⚠ **မွေးနံ ဘင်္ဂအိမ်ရောက်** → ပြိုလဲ/ပျက်စီးမှု သတိ |
| မရဏ | ပွဲ (၅) | ကြာသပတေး | ❌ ဆိုး | အားကောင်း | ✅ မရဏကို ကြာဖိနှိပ် — ဆိုးမှုလျော့ |
| အထွန်း | အောင် (၁) | တနင်္ဂနွေ | ✅ ကောင်း | အားကောင်း | ✅✅ ယခုနှစ် **ထွန်းပေါက်မှု အားကောင်းဆုံး** |
| သိုက် | လံ (၄) | ဗုဒ္ဓဟူး | ✅ ကောင်း | အားသင့် | ✅ ပညာ/ဆက်သွယ်ရေးမှ ဥစ္စာတိုး |
| ရာဇ | ထူ (၇) | စနေ | ✅ ကောင်း | အားဆိုး | ⚠ ရာထူးကိစ္စ နှေးကွေး/ဖိအား |
| ပုတိ | စစ် (၃) | အင်္ဂါ | ❌ ဆိုး | အားဆိုး | ⚠⚠ **ဆိုးဂြိုဟ် + ဆိုးအိမ်** = အထူးသတိ |
| အဓိပတိ | သူ (၆) | သောကြာ | ✅ ကောင်း | အားသင့် | ✅ ခေါင်းဆောင်နိုင်မှု + ကျေနပ်မှု |

### ၁၁.၇ မူလ + ကသစ် ပေါင်းဖတ်ခြင်း

**အချက် (က) — မွေးနံ ဘယ်အိမ်ရောက်သလဲ**

| | မူလခွင် | ကသစ်ခွင် (၁၃₈₈) | ပေါင်းဖတ် |
|---|---|---|---|
| တနင်္လာ | **အဓိပတိ** ✅ | **ဘင်္ဂ** ⚠ | မူလဇာတာကောင်းသော်လည်း ယခုနှစ် ဘင်္ဂရောက် — **အုတ်မြစ်ခိုင်ပေမယ့် မုန်တိုင်းခံရမည်** |

**အချက် (ခ) — ကသစ်ခွင်၏ အကောင်းဆုံး/အဆိုးဆုံး**

| အကောင်းဆုံး | အဆိုးဆုံး |
|---|---|
| တနင်္ဂနွေ → အထွန်း ✅✅ | အင်္ဂါ → ပုတိ ⚠⚠ |
| ခေါင်းဆောင်/ဂုဏ်သိက္ခာ နယ်ပယ်မှ ထွန်းပေါက်မှု | ကျန်းမာရေး/ရုပ်ပိုင်းဆိုင်ရာ **အထူးသတိ** |

**အချက် (ဂ) — မူလနှင့် ကသစ် အိမ်တူ/မတူ စစ်ဆေး**

| ဂြိုဟ် | မူလ၌ အိမ် | ကသစ်၌ အိမ် | ပြောင်းလဲချက် |
|---|---|---|---|
| တနင်္လာ | အဓိပတိ ✅ | ဘင်္ဂ ⚠ | ⬇ ကျဆင်း |
| တနင်္ဂနွေ | မရဏ ⚠ | အထွန်း ✅ | ⬆ တက်လာ |
| ကြာသပတေး | ဘင်္ဂ ✅ | မရဏ ✅ | → တည်ငြိမ် (ဆိုးအိမ်ကိုပဲ ဖိနှိပ်နေ) |
| ဗုဒ္ဓဟူး | အထွန်း ✅ | သိုက် ✅ | → ကောင်းဆက် |
| စနေ | သိုက် ⚠ | ရာဇ ⚠ | → ဖိအားပြောင်း |
| အင်္ဂါ | ရာဇ ⚠ | ပုတိ ⚠⚠ | ⬇ ပိုဆိုး — **ယတြာလို** |
| သောကြာ | ပုတိ ↔ | အဓိပတိ ✅ | ⬆ တက်လာ |

### ၁၁.₈ ပုံသေ ဖတ်နည်း အကျဉ်းချုပ်

```
အဆင့် ₁ — မူလခွင် ချ (မွေးနှစ် ÷ ₇)
           → ဘဝတစ်သက်တာ ဂြိုဟ်စီး ပုံသေ

အဆင့် ₂ — ကသစ်ခွင် ချ (ရောက်ဆဲနှစ် ÷ ₇)
           → ယခုနှစ် ဂြိုဟ်စီး ယာယီ

အဆင့် ₃ — မွေးနံ ကသစ်ခွင်၌ ဘယ်အိမ်ရောက်သလဲ
           → ယခုနှစ် ကံ ကောင်း/ဆိုး (အဓိက)

အဆင့် ₄ — ကသစ်ခွင်၏ ဂြိုဟ်စီး ₇ ခုလုံး ဖတ်
           → ယခုနှစ် ရာသီဥတု အသေးစိတ်

အဆင့် ₅ — မူလ + ကသစ် ပေါင်းဖတ်
           → "ဇာတာကောင်း + ယခုနှစ်ဆိုး" / "ဇာတာဆိုး + ယခုနှစ်ကောင်း"

အဆင့် ₆ — အသက်ပိုင်း × မူလဂြိုဟ်စီး
           → ဘဝ၏ ဘယ်ကာလ ဘာဖြစ်မည်

အဆင့် ₇ — ဘုမ္မိနှစ် ထပ်မိလျှင်
           → အထူးသတိ / ယတြာလိုအပ်
```

## ၁၂။ ၄၉ ဇာတာစုံ — ဖွား ၇ × အကြွင်း ၇

### ၁၂.၁ ၄၉ ဇာတာ ဖွဲ့စည်းပုံ

ဖွား ၇ မျိုး (အထွန်း/သိုက်/ရာဇ/အဓိပတိ/ပုတိ/ဘင်္ဂ/မရဏ) × အကြွင်း ၇ မျိုး (၀-၆) = **၄၉ ဇာတာ** ရှိသည်။

ထပ်၍ **နက္ခတ် ၃ မျိုး** ခွဲခြားနိုင်သေး—

| နက္ခတ် | သဘော |
|---|---|
| **လူနက္ခတ်** | သာမန်၊ လူ့ဘဝနှင့် နီးစပ် |
| **နတ်နက္ခတ်** | ကံကောင်း၊ နတ်ကူညီ |
| **ဘီလူးနက္ခတ်** | ကြမ်းတမ်း၊ ခက်ခဲ |

### ၁၂.၂ ဖွားဇာတာ အသေးစိတ် ဟောကိန်း (အကျဉ်းချုပ်)

#### ပုတိဖွား
- ငယ်ဇာတာ မကောင်း၊ ရောဂါထူပြော
- ကြီးလာရင် ပစ္စည်းဥစ္စာ ပြည့်စုံ
- တာဝန်ကျေပွန်လိုစိတ်ရှိ
- မကောင်းမှု၌ မွေ့လျော်တတ်သည် (သတိထားရန်)

#### ဘင်္ဂဖွား
- ကိုယ့်ဒူးကိုယ်ချွန် ကြိုးစာလိုစိတ်ရှိ
- ဆွေမျိုးထက် တစိမ်းက ပိုအကျိုးပေး
- ဇာတိအဝေးတွင် ထွန်းပေါက်
- ကောက်ရိုးမီးစိတ် — အစပြင်းပြီး နောက်ပိုင်းပျင်းလွယ်
- အလုပ်ခဏခဏ ပြောင်းမိတတ်

#### မရဏဖွား
- စိတ်ဓာတ်ပြင်းထန်၊ ထက်မြက်
- ဆုံးဖြတ်ချက်ချရာတွင် ချိတုံချတုံ
- အလွန်မေ့တတ်
- စေတနာကောင်းသော်လည်း ပြောစကားကြောင့် လူမုန်းများတတ်
- နှစ်ရှည်လုပ်ငန်းမသင့်၊ ပေါ်ပင်အလုပ်သင့်
- ဘဝအတတ်အကျများ၊ ငယ်ဇာတာမကောင်း၊ အိုဇာတာကောင်း

#### အထွန်းဖွား
- ခရီးကြိုက်၊ ပေါ့ပါးလွတ်လပ်သူ
- အိမ်မကပ်စီးပွားကောင်း
- ရုပ်ရည်ချော/လှတတ်

#### သိုက်ဖွား
- လက်တွေ့ကျ၊ စိတ်ကူးစိတ်သန်းဖြင့် ဖန်တီးလို့မရ
- ပင်ပန်းခံရမည့်ဇာတာ — သူများ ၁ နှစ်ကြိုးစားသမျှကို ၃ နှစ်ကြိုးစားရ
- ငွေကိုစနစ်တကျသုံးသော်လည်း မိသားစုကြောင့် ကုန်တတ်

#### ရာဇဖွား
- အာဏာ/ခေါင်းဆောင်ရာထူးရတတ်
- ဂုဏ်သိက္ခာကြီးမား
- ကံလက်ခံဂြိုဟ် အဆင့်မြင့်

#### အဓိပတိဖွား
- ဉာဏ်ရည်ထက်မြက်၊ သင်ပြရလွယ်
- ဇွဲလုံ့လ ပြင်းထန်
- ဆုံးဖြတ်ချက်ချပြီးရင် ပြန်ပြင်ခက်/ခေါင်းမာ
- အပေါ်ယံသိမျိုးများ — ဘာသာရပ်မျိုးစုံ နည်းနည်းစီလိုက်စား
- စကားပြောကြောင့် လူမုန်းများတတ်

---

## ၁၃။ ယတြာချေပနည်း

### ၁၃.၁ ယတြာ ဆိုသည်မှာ

ယတြာ = ဆိုးကျိုးကို ကာကွယ်/ချေဖျက်ရန်နှင့် ကောင်းကျိုးကို ပွားများရန် ပြုလုပ်သော ဗေဒင်ဆိုင်ရာ အမှုကိစ္စများဖြစ်သည်။ ဘုမ္မိကျသောနှစ်၊ ဂြိုဟ်ဆိုးစီးသောနှစ်တို့တွင် ယတြာချေပခြင်းသည် အထူးအရေးကြီးသည်။

### ၁၃.၂ ယတြာ အမျိုးအစားများ

#### (က) ဘာသာရေးယတြာ (အဓိက)
- **ဒါန** — ဆွမ်းကပ်ခြင်း၊ အလှူဒါနပြုခြင်း
- **သီလ** — ကိုးပါးသီလ/ဥပုသ်စောင့်ခြင်း
- **ဘာဝနာ** — တရားထိုင်ခြင်း၊ သမထ/ဝိပဿနာ
- **ပရိတ်တရားနာ** — မင်္ဂလာသုတ်၊ ရတနာသုတ်၊ မေတ္တာသုတ် စသည်
- **သက်သတ်လွှတ်** — ငါး/ငှက်/သတ္တဝါ လွှတ်ခြင်း

#### (ခ) နေ့နံယတြာ (နေ့သားအလိုက်)

| နေ့သား | ယတြာ |
|---|---|
| တနင်္ဂနွေ | ပျော့ရှပ်ရွက်/ကျွဲကောသီးဒါန၊ အဖြူရောင်ပစ္စည်းလှူ |
| တနင်္လာ | ရေ/ရေသန့်လှူ၊ အဖြူ/အဝါရောင်ပစ္စည်း |
| အင်္ဂါ | အနီရောင်ပစ္စည်းလှူ၊ ဆွမ်းအမျိုးမျိုးကပ် |
| ဗုဒ္ဓဟူး | အစိမ်းရောင်ပစ္စည်း၊ ပန်းလှူ၊ ဆွမ်းဟင်းလျာ |
| ကြာသပတေး | အဝါရောင်ပစ္စည်း၊ ဆီမီး/ဖယောင်းတိုင်ထွန်း |
| သောကြာ | အနီ/ပန်းရောင်ပစ္စည်း၊ ပန်းကပ် |
| စနေ | အမည်းရောင်ပစ္စည်း၊ ဆီမီးထွန်း |
| ရာဟု | မိမိဗုဒ္ဓဟူးနံယတြာအတိုင်း + နေ့လယ်ခင်းဆွမ်းကပ် |

#### (ဂ) ဘုမ္မိနှစ် အထူးယတြာ
- ရဟန်းကိုးပါး/တစ်ဆယ့်ခုနစ်ပါး ဆွမ်းကပ်
- ပရိတ်ကြီး ၁၁ သုတ် ရွတ်ဆိုကပ်
- ဘုရားကိုးဆူ ဖူးမျှော်
- ရေပူးကန်/ရေကန် ဖောက်ခြင်း
- တံတား/လမ်း ပြုပြင်ခြင်း
- အသက်ကြီးရဟန်းသံဃာအား ဆေးဝါးလှူဒါန်း

#### (ဃ) မဟာဘုတ်အိမ်အလိုက် ယတြာ

| အိမ် | အဆိုးကျရောက်လျှင် ယတြာ |
|---|---|
| ပုတိ | သက်သတ်လွှတ်၊ ဆေးလှူ၊ ရောဂါသည်ကူညီ |
| ဘင်္ဂ | ပြိုကျပျက်စီးရာ ပြုပြင်ရန် ကူညီ (ဘုရား/ကျောင်း ပြုပြင်ခြင်း) |
| မရဏ | ပရိတ်ရေသွန်း၊ မေတ္တာပို့၊ ဆွမ်းကပ်၊ အသုဘာနုဿတိ |

### ၁၃.၃ ယတြာ၏ အခြေခံမူ

1. **စေတနာသန့်ရမည်** — ယတြာ၏ အားသည် စေတနာအပေါ်တွင် မူတည်သည်
2. **ကံ ၃ ပါး ကောင်းရမည်** — ကိုယ်ကံ (ကာယကံ)၊ နှုတ်ကံ (ဝစီကံ)၊ စိတ်ကံ (မနောကံ)
3. **ယတြာသည် ကံကို ပြင်ခြင်းမဟုတ်** — ကံ၏ အားကို လျော့ပါးစေခြင်း/နောက်ဆက်တွဲ ကောင်းကံသစ် ဖန်တီးခြင်း
4. **ဗုဒ္ဓဘာသာအရ** — ကုသိုလ်ကံဖြင့် အကုသိုလ်ကံကို ဖုံးအုပ် (ဥပဃာတကကံ) ပြုခြင်း

## ၁၄။ ပက္ခတ် — အိမ်ထောင်ဖက် ဓာတ်တိုက်နည်း

### ၁၄.၁ ပက္ခတ် ဆိုသည်မှာ

ပက္ခတ် (Pakkhat) သည် လူနှစ်ဦး၏ **မွေးနံချင်း** ဓာတ်တိုက်ကြည့်၍ အိမ်ထောင်ဖက်/စီးပွားဖက်/မိတ်ဆွေ အဖြစ် **သင့်/မသင့်** ကြည့်သော နည်းလမ်းဖြစ်သည်။

### ၁၄.၂ တွက်နည်း

1. လင်မယား နှစ်ဦး၏ **မွေးနေ့ဂဏန်း** ကို ပေါင်းပါ
2. ပေါင်းလဒ်ကို **၇** ဖြင့် စား → အကြွင်းယူ
3. ကြွင်းပေါ်မူတည်၍ —

| အကြွင်း | အဓိပ္ပာယ် | အကျိုး |
|---|---|---|
| ၁ | ဗိဿနု (Vishnu) | အလွန်ကောင်း — ချမ်းသာကြွယ်ဝ |
| ၂ | ယမ (Yama) | ဆိုးရွား — ခွဲခွာခြင်း/ရန်ငြိုး |
| ၃ | သိကြား (Thagya) | ကောင်း — ရာထူးဂုဏ် |
| ၄ | မာရ်နတ် (Mara) | ဆိုး — စစ်ခိုက်/ဒုက္ခ |
| ၅ | ကုဝေရ (Kubera) | ကောင်း — ဥစ္စာပေါများ |
| ၆ | သိန္ဒု (Thindu) | ဆိုး — အကြမ်းဖက်ခံရ |
| ၀/၇ | ရာဟု (Rahu) | ဆိုးမကောင်း — အတက်အကျများ |

> **ဥပမာ** — တနင်္ဂနွေသား (၁) + သောကြာသမီး (၆) = ၇ ÷ ၇ = ၀ ကြွင်း = ရာဟု (သတိထားရန်)

---

## ၁၅။ ဓာတ်ရိုက်/ဓာတ်ဆင်

### ၁၅.၁ ဓာတ် ၄ ပါးနှင့် ဂြိုဟ်

| ဓာတ် | ပါဠိ | ဂြိုဟ် |
|---|---|---|
| **မြေဓာတ်** | ပထဝီ | စနေ၊ ကြာသပတေး |
| **ရေဓာတ်** | အာပေါ | တနင်္လာ၊ ဗုဒ္ဓဟူး |
| **မီးဓာတ်** | တေဇော | တနင်္ဂနွေ၊ အင်္ဂါ |
| **လေဓာတ်** | ဝါယော | သောကြာ |

### ၁၅.၂ ဓာတ်တည့်/မတည့် မူ

> **"မီးနှင့်ကား လေ၊ ရေနှင့်မြေ မိတ်ဆွေတော်ကြ သဟာယ"**

| ဓာတ်တွဲ | ဆက်ဆံရေး |
|---|---|
| မီး + လေ | **မိတ်** (မီးကို လေကအားပေး) |
| ရေ + မြေ | **မိတ်** (မြေကို ရေကအားပေး) |
| မီး + ရေ | **ရန်** (ရေက မီးကိုငြိမ်း) |
| မြေ + လေ | **ရန်** (ညှပ်ပိတ်/ချုပ်ခြင်း) |
| မီး + မြေ | **သမ** (ကြားနေ) |
| ရေ + လေ | **သမ** (ကြားနေ) |
| တူချင်း | **မိတ်/သမ** (အခြေအနေအလိုက်) |

### ၁၅.၃ ဓာတ်ရိုက်/ဓာတ်ဆင် ဆိုသည်မှာ

- **ဓာတ်ရိုက်** — အားနည်းနေသော ဓာတ်ကို ပြင်ပမှ အင်အားဖြည့်ပေးခြင်း (ယတြာမျိုး)
- **ဓာတ်ဆင်** — ဓာတ်တစ်ခုကို အခြားဓာတ်ဖြင့် ထိန်းချုပ်/ထောက်ကူခြင်း
- ဥပမာ — မီးဓာတ်ပြင်းသူ (အင်္ဂါသား) ကို ရေဓာတ် (တနင်္လာ) ဖြင့် ဓာတ်ဆင်ပေး

## ၁၉။ မြန်မာ့ရိုးရာ နာမည်ပေးနည်း (မွေးနေ့နံအလိုက်)

### ၂၀.၅ မဟာဘုတ်နှင့် နက္ခတ် — မည်သို့ ကွဲပြားသနည်း

## ၂၁။ မှတ်သားသင့်သော အချက်များ

1. မဟာဘုတ်သည် **အခြေခံဗေဒင်** ဖြစ်သော်လည်း နက်ရှိုင်းစွာ လေ့လာနိုင်သည်
2. အထက်တန်းဖွား ≠ ကံကောင်းသူ၊ အောက်တန်းဖွား ≠ ကံဆိုးသူ
3. **ကသစ်မဟာဘုတ်** ကို မူလမဟာဘုတ်နှင့် တွဲဖတ်မှ ယခုနှစ်ကံကို ပိုတိကျစွာ သိနိုင်
4. ဘုမ္မိနှစ်များကို **ဂြိုဟ်စီး** နှင့် **ကသစ်ဇာတာ** နှစ်ခုလုံးဖြင့် တွဲကြည့်ရန်
5. ယတြာချေပခြင်းသည် **စေတနာ** ပေါ်မူတည်၍ အမှန်တကယ် အကျိုးပေးနိုင်
6. မဟာဘုတ်ဟောကိန်းသည် **ယေဘုယျ** ဖြစ်၍ ပုဂ္ဂိုလ်တစ်ဦးချင်းစီ၏ ကံ/ကံ၏အကျိုး/ဝီရိယ တို့ကြောင့် ကွဲပြားနိုင်
7. **ပက္ခတ်** သည် အိမ်ထောင်ဖက်ရွေးချယ်ရာတွင် လမ်းညွှန်သာဖြစ်ပြီး တစ်ခုတည်းကို အဆုံးအဖြတ် မပြုသင့်
8. **ဆေးပညာမဟာဘုတ်** သည် ရိုးရာအသိပညာဖြစ်ပြီး ခေတ်သစ်ဆေးပညာနှင့် **အစားထိုးမရ**
9. **နာမည်ပေးခြင်း** သည် မြန်မာလူမျိုးတို့၏ ထူးခြားသော ယဉ်ကျေးမှုတစ်ရပ်
10. **နက္ခတ် ၂၇ လုံး** သည် Vedic စနစ်မှ ဆင်းသက်ပြီး မဟာဘုတ်နှင့် **မတူညီသော input** (မွေးချိန်) လိုအပ်သည်
11. ဗေဒင်/မဟာဘုတ်ပညာသည် **ယုံကြည်မှု** အပေါ်အခြေခံ — မြန်မာ့ယဉ်ကျေးမှု အမွေအနှစ်အဖြစ် တန်ဖိုးထား၍ လေ့လာသင့်
